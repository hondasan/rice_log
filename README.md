# 🍚 こめログ (rice_log)

自宅のお米の在庫残量を「あと何日分もつか」で把握できる、SPA（シングルページアプリケーション）構成の静的Webアプリです。

## ✨ 特徴

- **キーボード入力ゼロのUX**: 合数や重量、銘柄の入力など、すべての操作がタップやスワイプのみで完結。スマホでの入力ストレスを徹底的に排除しました。
- **直感的な米びつビジュアル**: 残量％に応じて中身の量やグラデーションが変化する Canvas 米びつインジケーターを搭載。
- **まとめて入力**: 旅行などで炊飯記録を数日分サボってしまった場合でも、カレンダーで日付を選ぶだけで一括記録が可能。
- **インライン操作のカレンダー**: 履歴画面はモーダルを一切開かず、インラインでの編集・スワイプ削除に対応。
- **お米管理＆使い切り**: 開封中のお米とストック（未開封）を別々に管理。お米を使い切った際、残っていた推定量を「食べきった」か「廃棄した」かを選択して記録し、次のストックを即時開封するフローを搭載。
- ** Canvas による統計グラフ**: 週別の棒グラフや年間推移の折れ線グラフを自前 Canvas 2D API でスムーズに描画。
- **設定とデータバックアップ**: ワンタップ炊飯量やテーマ（ライト/ダーク）の切り替え。データのエクスポート/インポート（JSON形式）、誤削除を防ぐための3秒長押し形式のデータ初期化機能を搭載。

## 🎨 デザインシステム

- **コンセプト**: 「台所の黒板と木のぬくもり」をテーマに、AI生成感を排除したナチュラルな風合い。
- **フォント**: 丸みのある温かい「Zen Maru Gothic」と言Outfit（数字用）を Google Fonts からインポートして使用。
- **カラー**: 生成り背景 (`#FAF6F0`)、抹茶グリーン (`#7B9E6B`)、稲穂ゴールド (`#D4A76A`) の日本の食卓に馴染む色合い。

## 🛠 実行方法

本アプリは外部フレームワークやビルドツールを使用しない純粋な HTML + CSS + JS の静的Webアプリです。
ディレクトリ配下のファイルを静的ホスティング（Vercel, Netlify, GitHub Pages など）にアップロードするか、ローカル環境で簡易的なHTTPサーバーを起動して `index.html` にアクセスすることで動作します。

**ローカルでの簡易起動例 (Node.js)**
```bash
# http-serverなどを利用して起動
npx http-server -p 8000
```
または Python で起動：
```bash
python3 -m http.server 8000
```
ブラウザで `http://localhost:8000` を開くことでご利用いただけます。
